# Model Context Protocol (MCP)

The **Model Context Protocol** is an open standard for connecting AI models to external tools, data sources, and services. It defines a structured JSON-RPC interface that lets any MCP-compatible client discover and invoke tools exposed by an MCP server -- without custom API integration.

PURECORTEX implements an MCP server that currently exposes a local decision-node tool for tri-brain consensus. External AI agents and IDE extensions can invoke it through a standardized MCP interface.

<Note>
  PURECORTEX implements **MCP 1.0** using the [FastMCP](https://github.com/jlowin/fastmcp) Python framework.
</Note>

## PURECORTEX as a Decision Node

In an MCP network, PURECORTEX operates as a **Decision Node** -- a composable intelligence endpoint that other agents can invoke to get validated, consensus-backed responses. Instead of calling a single LLM, callers get the output of parallel Claude Opus 4.6 + Gemini 2.5 Pro + GPT-5 inference, filtered through the protocol's permission sandbox.

```
External Agent ──► MCP Client ──► PURECORTEX MCP Server
                                       │
                                       ├── ConsensusOrchestrator
                                       │     ├── Claude Opus 4.6
                                       │     ├── Gemini 2.5 Pro
                                       │     └── GPT-5
                                       │
                                       └── PermissionProxy (validation)
```

Every tool invocation passes through the `PermissionProxy` before a response is returned. The MCP server runs at the `READ_ONLY` permission tier by default, meaning it can respond to queries but cannot trigger on-chain actions.

## Implementation

The MCP server is defined in `backend/mcp_server.py` and uses the FastMCP framework:

```python
from mcp.server.fastmcp import FastMCP
from orchestrator import ConsensusOrchestrator
from sandboxing import PermissionProxy, PermissionTier

mcp = FastMCP("PURECORTEX", description="PURECORTEX Tri-Brain Orchestrator MCP Server")
proxy = PermissionProxy(PermissionTier.READ_ONLY)
orchestrator = ConsensusOrchestrator()
```

To start the server locally:

```bash
cd backend
python mcp_server.py
```
