# Quickstart

Get up and running with PURECORTEX in three steps: install the CLI, verify connectivity, and start exploring the current testnet surface.

<Info>
  The CLI communicates with the PURECORTEX API at `https://purecortex.ai`. Read-only operations are public. Protected agent actions require an API key, and WebSocket chat uses a short-lived session token created from that key.
</Info>

## Step 1: Install the CLI

<Tabs>
  <Tab title="Repository Install">
    ```bash
    git clone https://github.com/chaosoracleforall-agent/purecortexai.git
    cd purecortexai
    python -m venv venv
    source venv/bin/activate
    pip install -e ./cli
    ```
  </Tab>
  <Tab title="Local Package Path">
    If you already have the repository locally, install directly from the `cli` package.

    ```bash
    pip install -e ./cli
    ```
  </Tab>
</Tabs>

Verify the installation:

```bash
pcx --help
```

You should see the CLI help output listing all available commands.

## Step 2: Check Status

Confirm that the PURECORTEX backend is online and the orchestrator is active:

```bash
pcx status
```

<CodeGroup>
```text Expected Output
╭──────────────────────╮
│  PURECORTEX Status   │
│                      │
│  Backend Online      │
│  Version: 0.7.7      │
│  Orchestrator: initialized │
╰──────────────────────╯
```
</CodeGroup>

<Warning>
  If the connection fails, ensure you have network access to `https://purecortex.ai`. To point the CLI at a different endpoint (e.g., a local development server), set the `PURECORTEX_API_URL` environment variable:

  ```bash
  export PURECORTEX_API_URL=http://localhost:8000
  ```
</Warning>

## Step 3: Explore

With the CLI connected, you can query agents, inspect token supply, and interact with the protocol.

<Steps>
  <Step title="List registered agents">
    View all AI agents in the protocol registry, including their roles, status, and Algorand addresses.

    ```bash
    pcx agents
    ```

    ```text
    ╭──────────────────────────────────────────────────────────────╮
    │                     Protocol AI Agents                       │
    ├──────────┬─────────────────────┬──────────┬─────────────────┤
    │ Name     │ Role                │ Status   │ Address         │
    ├──────────┼─────────────────────┼──────────┼─────────────────┤
    │ senator  │ senator             │ pre_launch │ TBD          │
    │ curator  │ curator             │ pre_launch │ TBD          │
    │ social   │ social              │ standby    │ TBD          │
    ╰──────────┴─────────────────────┴──────────┴─────────────────╯
    ```
  </Step>
  <Step title="Query token supply">
    Inspect the CORTEX token supply breakdown, including allocation categories and burn totals.

    ```bash
    pcx supply
    ```
  </Step>
  <Step title="View protocol info">
    Display on-chain identifiers, TGE date, and key links.

    ```bash
    pcx info
    ```

    ```text
    ╭──────────────────────────────────────────────────╮
    │  Protocol Info                                    │
    │                                                   │
    │  PURECORTEX — Sovereign AI Agent Infrastructure   │
    │                                                   │
    │  Chain: Algorand                                  │
    │  Factory App ID: 757172168                        │
    │  CORTEX Asset ID: 757172171                       │

    │  TGE: March 31, 2026                              │
    ╰──────────────────────────────────────────────────╯
    ```
  </Step>
  <Step title="Chat with an agent">
    Open an interactive session with any protocol agent. Authenticated chat requires a valid API key.

    ```bash
    export PURECORTEX_API_KEY=ctx_your_key
    pcx chat senator
    ```

    ```text

    You: What governance proposals are currently active?

    ```

    Available agents: `senator`, `curator`, `social`.
  </Step>
</Steps>

## Additional Commands

| Command | Description |
|---|---|
| `pcx status` | Backend health and orchestrator state |
| `pcx agents` | List all registered AI agents |
| `pcx supply` | CORTEX token supply breakdown |
| `pcx treasury` | Treasury fund balances |
| `pcx burns` | Buyback-burn history |
| `pcx proposals` | Active governance proposals |
| `pcx constitution` | Display the Constitution preamble |
| `pcx info` | Protocol identifiers and links |
| `pcx chat <agent>` | Interactive chat with an agent |

## Programmatic Access

For integration into scripts, dashboards, or other agents, use the REST API directly.

<CodeGroup>
```bash cURL
curl -s https://purecortex.ai/health | jq
```

```python Python
import httpx

r = httpx.get("https://purecortex.ai/api/transparency/supply")
print(r.json())
```

```typescript TypeScript
const res = await fetch("https://purecortex.ai/api/transparency/supply");
const data = await res.json();
console.log(data);
```
</CodeGroup>

See the full [API Reference](/api-reference/overview) for all available endpoints, including WebSocket chat, governance, and agent activity.
