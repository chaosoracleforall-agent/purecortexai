# SDK Overview

PURECORTEX now includes first-party SDK packages in this repository for Python and TypeScript/JavaScript. They wrap the live REST API, authenticated chat-session bootstrap flow, and governance/transparency reads so new integrations can start with an official client instead of raw `fetch` or `httpx` snippets.

<Info>
  Registry publishing is still on the roadmap. Today, install the packages from the repository or a local checkout.
</Info>

## Quick Start: Python SDK

```python
from purecortex_sdk import PureCortexClient

with PureCortexClient(api_key="ctx_your_key") as client:
    health = client.health()
    print(health["status"])

    supply = client.supply()
    print(supply["total_supply"])

    reply = client.chat("senator", "Summarize the governance framework.")
    print(reply["response"])
```

---

## Quick Start: TypeScript SDK

```typescript
import { PureCortexClient } from "@purecortex/sdk";

const client = new PureCortexClient({ apiKey: "ctx_your_key" });

const health = await client.health();
console.log(health.status);

const overview = await client.governanceOverview();
console.log(overview.total_proposals);

const reply = await client.chat("social", "Give me a positioning summary.");
console.log(reply.response);
```

## Covered Features

- Health and dependency status
- Transparency: supply, treasury, burns, governance, live agent ASAs
- Agent registry and agent activity
- Authenticated agent chat
- Chat-session bootstrap and WebSocket URL helpers
- Governance overview, proposal listing, proposal detail, and on-chain reads
- Governance writes and admin key-management helpers
