# TypeScript SDK

This guide covers the official TypeScript/JavaScript SDK in `sdk/typescript`. It wraps the live REST API, authenticated chat-session bootstrap flow, and governance/transparency reads. Use raw `algosdk` only when you need direct on-chain access outside the HTTP API surface.

## Installation

```bash
npm install ./sdk/typescript
```

<Info>
  The package is installable from a local checkout today. Registry publication to npm is still on the roadmap.
</Info>

---

## Quick Start

```typescript
import { PureCortexClient } from "@purecortex/sdk";

const client = new PureCortexClient({ apiKey: "ctx_your_key" });

const health = await client.health();
console.log(health.status);

const supply = await client.supply();
console.log(supply.total_supply);

const reply = await client.chat("social", "Give me a positioning summary.");
console.log(reply.response);
```

## Current Client Methods

- `health()`
- `supply()`, `treasury()`, `burns()`, `governanceTransparency()`, `transparencyAgents()`
- `listAgents()`, `agentActivity(agentName)`, `chat(agentName, message)`
- `constitution()`, `governanceOverview()`, `listProposals()`, `proposal(id)`, `onchainProposals()`
- `createProposal(...)`, `reviewProposal(...)`, `vote(...)`
- `createChatSession()`, `websocketUrl(sessionToken)`, `connectChat()`
- `bootstrapAdminKey(...)`, `createApiKey(...)`, `revokeApiKey(...)`

## WebSocket Chat

Real-time bidirectional chat with PURECORTEX AI agents.

<Tabs>
  <Tab title="Node.js">
    ```typescript
    import { PureCortexClient } from "@purecortex/sdk";

    const client = new PureCortexClient({
      apiKey: process.env.PURECORTEX_API_KEY!,
      WebSocketImpl: WebSocket,
    });

    const ws = await client.connectChat();

    ws.addEventListener("open", () => {
      console.log("Connected to PURECORTEX");
      ws.send("What is the CORTEX token used for?");
    });

    ws.addEventListener("message", (event) => {
      console.log(`Agent: ${event.data}`);

      // Send a follow-up
      ws.send("How does veCORTEX staking work?");
    });

    ws.addEventListener("close", () => {
      console.log("Disconnected");
    });
    ```
  </Tab>
  <Tab title="Browser">
    ```typescript
    import { PureCortexClient } from "@purecortex/sdk";

    const client = new PureCortexClient({
      apiKey: window.sessionStorage.getItem("purecortex_api_key")!,
    });

    const ws = await client.connectChat();

    ws.onopen = () => {
      console.log("Connected");
      ws.send("Explain tri-brain consensus.");
    };

    ws.onmessage = (event: MessageEvent) => {
      const agentResponse = event.data;
      document.getElementById("chat")!.innerText += `\nAgent: ${agentResponse}`;
    };

    ws.onerror = (error) => {
      console.error("WebSocket error:", error);
    };
    ```
  </Tab>
</Tabs>

<Warning>
  WebSocket messages are limited to 4,096 characters. Messages exceeding this limit are rejected with an error response.
</Warning>

<Note>
  If your runtime does not expose a global `WebSocket`, inject one with `WebSocketImpl`.
</Note>

## Algorand On-Chain Reads

Use `algosdk` to query smart contract state and agent token data directly from the Algorand indexer.

### Query the Algorand Indexer

```typescript
import algosdk from "algosdk";

const FACTORY_APP_ID = 757172168;
const CORTEX_ASSET_ID = 757172171;
const INDEXER_URL = "https://testnet-idx.algonode.cloud";

const indexer = new algosdk.Indexer("", INDEXER_URL, "");

async function getFactoryInfo() {
  const appInfo = await indexer.lookupApplications(FACTORY_APP_ID).do();
  const app = appInfo.application;

  console.log(`App ID: ${app.id}`);
  console.log(`Creator: ${app.params.creator}`);

  // Read global state
  const globalState = app.params["global-state"] || [];
  for (const state of globalState) {
    const key = Buffer.from(state.key, "base64").toString();
    console.log(`  ${key}: ${JSON.stringify(state.value)}`);
  }

  return appInfo;
}
```

### Query CORTEX Asset

```typescript
async function getCortexInfo() {
  const assetInfo = await indexer.lookupAssetByID(CORTEX_ASSET_ID).do();
  const asset = assetInfo.asset;
  const params = asset.params;

  console.log(`Asset ID:  ${asset.index}`);
  console.log(`Name:      ${params.name}`);
  console.log(`Unit:      ${params["unit-name"]}`);
  console.log(`Total:     ${params.total.toLocaleString()}`);
  console.log(`Decimals:  ${params.decimals}`);
  console.log(`URL:       ${params.url || "N/A"}`);

  return assetInfo;
}
```

### Search for Agent Tokens

```typescript
async function findAgentTokens() {
  const txns = await indexer
    .searchForTransactions()
    .applicationID(FACTORY_APP_ID)
    .txType("acfg")
    .do();

  const agentIds: number[] = [];
  for (const txn of txns.transactions || []) {
    const acfg = txn["asset-config-transaction"];
    if (acfg && acfg["created-asset-index"]) {
      agentIds.push(acfg["created-asset-index"]);
      console.log(`Agent token found: ASA ${acfg["created-asset-index"]}`);
    }
  }

  return agentIds;
}
```

## Full Example

```typescript
import algosdk from "algosdk";

const BASE_URL = "https://purecortex.ai";
const INDEXER_URL = "https://testnet-idx.algonode.cloud";
const CORTEX_ASSET_ID = 757172171;

async function main() {
  // 1. REST: Get supply info
  const supply = await fetch(`${BASE_URL}/api/transparency/supply`)
    .then((r) => r.json());
  console.log(`CORTEX Total Supply: ${supply.total_supply.toLocaleString()}\n`);

  // 2. REST: List agents
  const registry = await fetch(`${BASE_URL}/api/agents/registry`)
    .then((r) => r.json());
  for (const agent of registry.agents) {
    console.log(`Agent: ${agent.name} (${agent.status})`);
  }

  // 3. REST: Chat with Senator
  const chat = await fetch(`${BASE_URL}/api/agents/senator/chat`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key": process.env.PURECORTEX_API_KEY!,
    },
    body: JSON.stringify({ message: "Summarize the governance framework." }),
  }).then((r) => r.json());
  console.log(`\nSenator: ${chat.response}`);

  // 4. On-chain: Query CORTEX asset
  const indexer = new algosdk.Indexer("", INDEXER_URL, "");
  const asset = await indexer.lookupAssetByID(CORTEX_ASSET_ID).do();
  console.log(`\nOn-chain CORTEX decimals: ${asset.asset.params.decimals}`);
}

main().catch(console.error);
```
