---
title: "List Proposals"
api: "GET /api/governance/proposals"
description: "Retrieve all governance proposals with their current status and vote tallies."
---

Returns a list of all governance proposals, including status, vote counts, and quorum information.

<ResponseField name="total" type="integer" required>
  Total number of proposals returned.
</ResponseField>

<ResponseField name="proposals" type="array" required>
  Array of proposal summaries.
  <Expandable title="proposal properties">
    <ResponseField name="id" type="string" required>
      Unique proposal identifier.
    </ResponseField>
    <ResponseField name="title" type="string" required>
      Proposal title.
    </ResponseField>
    <ResponseField name="status" type="string" required>
      Current proposal status. One of: `discussion`, `voting`, `passed`, `rejected`, `executed`, `cancelled`.
    </ResponseField>
    <ResponseField name="proposer" type="string" required>
      Algorand address of the proposal author.
    </ResponseField>
    <ResponseField name="created_at" type="string" required>
      ISO 8601 timestamp of when the proposal was created.
    </ResponseField>
    <ResponseField name="voting_ends" type="string" required>
      ISO 8601 timestamp of when the voting period ends. Empty string if not yet in voting.
    </ResponseField>
    <ResponseField name="votes_for" type="number" required>
      Total veCORTEX weight voting in favor.
    </ResponseField>
    <ResponseField name="votes_against" type="number" required>
      Total veCORTEX weight voting against.
    </ResponseField>
    <ResponseField name="quorum_reached" type="boolean" required>
      Whether the proposal has reached the required quorum threshold.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash curl
curl https://purecortex.ai/api/governance/proposals
```
</RequestExample>
