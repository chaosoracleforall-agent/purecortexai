---
title: "Health Check"
api: "GET /health"
description: "Check the API server status and version."
---

Returns the current health status, version, and dependency state of the PURECORTEX API.

<ResponseField name="status" type="string" required>
  Server status. Returns `"ok"` when core dependencies are healthy and `"degraded"` when a dependency such as Redis is unavailable.
</ResponseField>

<ResponseField name="version" type="string" required>
  Current API version.
</ResponseField>

<ResponseField name="dependencies" type="object" required>
  Current dependency state for Redis, the orchestrator, and the agent loop.
</ResponseField>

<RequestExample>
```bash curl
curl https://purecortex.ai/health
```
</RequestExample>

<ResponseExample>
```json 200
{
  "status": "ok",
  "version": "0.7.0",
  "dependencies": {
    "redis": "connected",
    "orchestrator": "initialized",
    "agent_loop": "running"
  }
}
```
</ResponseExample>
