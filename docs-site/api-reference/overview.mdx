## Base URL

All API requests are made to:

```
https://purecortex.ai
```

## Authentication

Read-only endpoints (transparency, governance, agents) are **public** and require no authentication.

Protected REST endpoints require an `X-API-Key`. WebSocket chat requires a short-lived session token minted by `POST /api/chat/session`.

## Rate Limits

| Channel        | Limit           |
|----------------|-----------------|
| REST API       | 60 req/min      |
| WebSocket      | 10 msg/min      |
| Transparency   | 120 req/min     |

Rate-limited responses return HTTP `429` with a `Retry-After` header indicating seconds until the next permitted request.

## Response Format

All endpoints return JSON with **snake_case** field names.

Successful responses return the data directly:

```json
{
  "status": "ok",
  "version": "0.7.0"
}
```

Error responses include `error` and `detail` fields:

```json
{
  "error": "not_found",
  "detail": "Proposal with the given ID does not exist."
}
```

## Error Codes

| HTTP Code | Meaning                | Description                                          |
|-----------|------------------------|------------------------------------------------------|
| `200`     | OK                     | Request succeeded.                                   |
| `400`     | Bad Request            | Malformed request body or missing required fields.   |
| `401`     | Unauthorized           | Missing or invalid API key or chat session. |
| `404`     | Not Found              | The requested resource does not exist.               |
| `429`     | Too Many Requests      | Rate limit exceeded. Retry after the indicated delay. |
| `500`     | Internal Server Error  | Unexpected server-side failure.                      |
| `503`     | Service Unavailable    | The service is temporarily down for maintenance.     |
