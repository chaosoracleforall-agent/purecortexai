---
title: "Get Burns"
api: "GET /api/transparency/burns"
description: "Retrieve the total burned supply and full burn transaction history."
---

Returns the cumulative amount of CORTEX tokens burned and a chronological history of each burn event.

<ResponseField name="total_burned" type="number" required>
  Total CORTEX tokens burned to date.
</ResponseField>

<ResponseField name="burn_history" type="array" required>
  Array of individual burn transactions, ordered by most recent first.
  <Expandable title="burn_history item properties">
    <ResponseField name="txn_id" type="string" required>
      Algorand transaction ID for the burn.
    </ResponseField>
    <ResponseField name="amount" type="number" required>
      Number of CORTEX tokens burned in this transaction.
    </ResponseField>
    <ResponseField name="round" type="integer" required>
      Algorand round (block number) in which the burn was confirmed.
    </ResponseField>
    <ResponseField name="timestamp" type="string" required>
      ISO 8601 timestamp of the burn transaction.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="note" type="string">
  Optional note with additional context about the burn mechanism.
</ResponseField>

<RequestExample>
```bash curl
curl https://purecortex.ai/api/transparency/burns
```
</RequestExample>
