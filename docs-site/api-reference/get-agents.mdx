---
title: "Get Agents (Transparency)"
api: "GET /api/transparency/agents"
description: "Retrieve agent factory information and the list of deployed AI agents on Algorand."
---

Returns the on-chain agent factory details and all deployed AI agents, including their Algorand Standard Asset (ASA) metadata.

<ResponseField name="factory_app_id" type="integer" required>
  Algorand Application ID of the agent factory smart contract. Current value: `757172168`.
</ResponseField>

<ResponseField name="total_agents" type="integer" required>
  Total number of AI agents deployed through the factory.
</ResponseField>

<ResponseField name="agents" type="array" required>
  Array of deployed agent records.
  <Expandable title="agent properties">
    <ResponseField name="asset_id" type="integer" required>
      Algorand Standard Asset (ASA) ID representing this agent.
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Human-readable agent name.
    </ResponseField>
    <ResponseField name="unit_name" type="string" required>
      Short unit name for the agent's ASA token.
    </ResponseField>
    <ResponseField name="total" type="number" required>
      Total supply of the agent's ASA token.
    </ResponseField>
    <ResponseField name="decimals" type="integer" required>
      Number of decimal places for the agent's ASA token.
    </ResponseField>
    <ResponseField name="creator_txn" type="string" required>
      Algorand transaction ID that created this agent's ASA.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="note" type="string">
  Optional note about the agent factory or deployment status.
</ResponseField>

<RequestExample>
```bash curl
curl https://purecortex.ai/api/transparency/agents
```
</RequestExample>
