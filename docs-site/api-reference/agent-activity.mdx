---
title: "Agent Activity"
api: "GET /api/agents/{agent_name}/activity"
description: "Retrieve the recent activity log for a specific PureCortex AI agent."
---

Returns the activity history for a given agent, including on-chain transactions and off-chain actions.

<ParamField path="agent_name" type="string" required>
  The agent whose activity to retrieve. One of: `senator`, `curator`, or `social`.
</ParamField>

<ResponseField name="agent" type="string" required>
  Name of the agent.
</ResponseField>

<ResponseField name="total_actions" type="integer" required>
  Total number of recorded actions for this agent.
</ResponseField>

<ResponseField name="recent_activity" type="array" required>
  Array of recent activity entries, ordered most recent first.
  <Expandable title="activity item properties">
    <ResponseField name="action" type="string" required>
      Short identifier for the action type (e.g., `"proposal_review"`, `"content_publish"`, `"social_post"`).
    </ResponseField>
    <ResponseField name="detail" type="string" required>
      Human-readable description of what the agent did.
    </ResponseField>
    <ResponseField name="timestamp" type="string" required>
      ISO 8601 timestamp of when the action occurred.
    </ResponseField>
    <ResponseField name="txn_id" type="string">
      Algorand transaction ID, if the action was recorded on-chain. `null` for off-chain actions.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash curl
curl https://purecortex.ai/api/agents/senator/activity
```
</RequestExample>
