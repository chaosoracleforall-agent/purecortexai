---
title: "Get Governance Stats"
api: "GET /api/transparency/governance"
description: "Retrieve high-level governance statistics via the transparency endpoint."
---

Returns aggregate governance metrics including proposal counts, voter participation, and total veCORTEX locked. This is the transparency-oriented governance summary.

<ResponseField name="total_proposals" type="integer" required>
  Total number of governance proposals ever created.
</ResponseField>

<ResponseField name="active_proposals" type="integer" required>
  Number of proposals currently in an active state (discussion or voting).
</ResponseField>

<ResponseField name="participation_rate" type="number" required>
  Average voter participation rate as a decimal (e.g., `0.42` for 42%).
</ResponseField>

<ResponseField name="total_vecortex" type="number" required>
  Total veCORTEX tokens locked for governance voting power.
</ResponseField>

<ResponseField name="note" type="string">
  Optional note about the governance system status.
</ResponseField>

<RequestExample>
```bash curl
curl https://purecortex.ai/api/transparency/governance
```
</RequestExample>

<ResponseExample>
```json 200
{
  "total_proposals": 12,
  "active_proposals": 2,
  "participation_rate": 0.42,
  "total_vecortex": 850000000000000,
  "note": "Governance is currently in the initial bootstrap phase."
}
```
</ResponseExample>
