# Brand Manual: PureCortex 🦞

## **1. The Identity**
**PureCortex** represents the fusion of mathematical certainty (Algorand) and high-fidelity intelligence (Dual-Brain AI). It is the premier platform for autonomous agentic commerce.

## **2. Color Palette**
| Component | Hex Code | Purpose |
| :--- | :--- | :--- |
| **Obsidian** | `#050505` | Background / Foundation |
| **Neural Blue** | `#007AFF` | Primary Action / Accent / Intelligence |
| **Pure White** | `#FFFFFF` | Primary Content / Clarity |
| **Graphite** | `#1A1A1A` | Secondary Backgrounds / Containers |

## **3. Typography**
*   **Primary (UI/Marketing):** **Inter** (Bold/ExtraBold).
    *   Used for headlines and primary brand text.
*   **Secondary (Content):** **Inter** (Regular/Medium).
    *   Used for body copy and descriptions.
*   **Technical (Data):** **JetBrains Mono**.
    *   Used for hashes, addresses, and agent terminal logs.

## **4. Iconography**
The **Atomic Neuron** icon should always maintain its clear geometric proportions. 
- The outer ring represents the network (Algorand).
- The inner synapse represents the core intelligence.
- The central point represents the singularity of the agent.

### **4.1 $CORTEX Token Icon**
The token icon uses a **Neural Blue gradient** (#007AFF to #003F7F) with a simplified white Atomic Neuron. It should be used for all on-chain metadata and wallet displays.

### **4.2 Twitter/X Profile Picture**
The PFP uses the high-contrast **Obsidian background** (#050505) with the full-glow Atomic Neuron. This projects a sense of deep-infrastructure stability and "Always Online" autonomy.

## **5. Usage Rules**
- Always maintain a clear margin around the logo (min 20% of width).
- Do not distort the aspect ratio.
- Preferred background is Obsidian (#050505) or deep gradients.
